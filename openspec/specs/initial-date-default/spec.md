## ADDED Requirements

### Requirement: Default Wedding Start Date
The booking form SHALL default the initial selection for both Arriving and Departing dates to the first day of the wedding event, December 17, 2026.

#### Scenario: Verify initial date values
- **WHEN** a user navigates to the booking form section
- **THEN** the "Arriving date" and "Departing date" fields are pre-filled with "2026-12-17".
